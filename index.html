1.Apa yang anda harapkan dari MK basis data

Saya berharap dari mata kuliah basis data bisa memahami cara menyimpan, mengelola, dan mengolah data dengan baik. Selain itu, 
saya ingin menguasai penggunaan SQL untuk operasi CRUD, 
serta mampu merancang database yang efisien agar bisa digunakan dalam pembuatan aplikasi nyata dan berguna di dunia kerja

2.Apa itu Basis Data

Basis data adalah kumpulan data yang terorganisir secara sistematis agar mudah diakses, dikelola, dan diperbarui
Database menyimpan informasi dalam bentuk tabel yang saling berhubungan sehingga memudahkan pengguna untuk mencari dan memanipulasi data

3.Apa itu Conceptual DB

Conceptual Database dapat dipahami sebagai gambaran umum database yang menjelaskan entitas serta keterkaitannya 
namun belum membahas detail implementasi.
untuk mempermudah perancangannya, model ini sering divisualisasikan dengan Entity Relationship Diagram (ERD) 
sebelum masuk tahap pembangunan fisik.

4.Apa itu Physical model

Physical model adalah representasi database pada level implementasi nyata yang menjelaskan bagaimana data disimpan dalam sistem manajemen basis data (DBMS). 
Model ini mencakup detail teknis seperti tipe data, struktur tabel, indeks, partisi, dan cara penyimpanan fisik di harddisk/server
